== Description ==

Aahana is lightweight, highly extendable and multi-purpose WordPress Theme, it will enable you to create almost any type of website with a beautiful & professional design. Aahana supports popular WordPress plugins such as Elementor, WPML, Polylang, Yoast SEO, WooCommerce, Contact Form 7, Jetpack, and much more. Aahana Pro demo https://preview.desertthemes.com/pro/aahana/


== Installation ==
	
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Aahana in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
